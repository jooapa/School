Kapselointi

Kapselointi tarkoittaa ohjelmoinnissa yhteen kuuluvien tietojen ja toimintojen kokoamista yhdeksi kokonaisuudeksi.
Olio-ohjelmoinnissa kapselointi tapahtuu usein kokoamalla muuttujat ja metodit luokaksi.

class Juna{
 aseta(malli, vuosi){
  this.malli = malli
  this.vuosi = vuosi
 }
 otaMalli(){
  console.log(this.malli)
 }
 otaVuosi(){
  console.log(this.vuosi)
 }
 const j = new Juna
 j.aseta("vr-juna", 2010) //asettaa nämä tiedot
 j.otaMalli()
 j.otavuosi()
 //Tulostaa konsoliin /"vr juna" "2010"\
}

Periytyminen

Periytyminen tarkoittaa, että jonkin ns. kantaluokan (base class) ominaisuuksia voidaan
periyttää (inherit) kantaluokasta johdetulle luokalle (derived class).

class Animal {
constructor(name) {
this.speed = 0;
this.name = name;
}
run(speed) {
this.speed = speed;
alert(`${this.name} runs with speed ${this.speed}.`);
}
stop() {
this.speed = 0;
alert(`${this.name} stands still.`);
}
}

let animal = new Animal("My animal ");

class Rabbit extends Animal {
hide() {
alert(`${this.name} hides!`);
}
}

let rabbit = New Rabbit("White Rabbit");

rabbit.run(5); // White Rabbit runs with speed 5.
rabbit.hide(); // White Rabbit hides!
