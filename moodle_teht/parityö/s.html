Konsruktorit

Javascriptissä luokka määritellään konstruktorifunktion avulla. Luodaan siis uusi funktio, jolla kaikki luokan uudet alkiot alustetaan.
Konstruktorin nimi on sama kuin luokan nimi

Kapselointi

Kapselointi tarkoittaa ohjelmoinnissa yhteen kuuluvien tietojen ja toimintojen kokoamista yhdeksi kokonaisuudeksi.
Olio-ohjelmoinnissa kapselointi tapahtuu usein kokoamalla muuttujat ja metodit luokaksi.

class Juna{
 aseta(malli, vuosi){
  this.malli = malli
  this.vuosi = vuosi
 }
 otaMalli(){
  console.log(this.malli)
 }
 otaVuosi(){
  console.log(this.vuosi)
 }
 const j = new Juna
 j.aseta("vr-juna", 2010) //asettaa nämä tiedot
 j.otaMalli()
 j.otavuosi()
 //Tulostaa konsoliin /"vr juna" "2010"\
}

Periytyminen

Periytyminen tarkoittaa, että jonkin ns. kantaluokan (base class) ominaisuuksia voidaan
periyttää (inherit) kantaluokasta johdetulle luokalle (derived class).

var a = {
x: 10,
calculate: function (z) {
return this.x + this.y + z
}
};

var b = {
y: 20,
__proto__: a
 };

var c = {
y: 30,
__proto__: a
 };

// call the inherited method
 b.calculate(30); // 60
 c.calculate(40); // 80
